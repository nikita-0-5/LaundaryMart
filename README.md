# 🧺 Laundry Wallah / Laundry Mart

A simple HTML-based laundry service website created using basic HTML elements. This project showcases a laundry business landing page where users can view available services, check pricing, and submit a booking request.

## 📌 Project Overview

Laundry Wallah / Laundry Mart is a beginner-friendly web project designed to demonstrate the use of:

- Semantic HTML structure
- Navigation links
- Images
- Lists
- Tables
- Forms
- Sections and Footer

The website provides information about laundry services and allows customers to submit their details for booking.

---

## 🚀 Features

### 🏠 Home Section
- Welcoming heading and business introduction.
- Brief description of the laundry service.

### 🔗 Navigation Menu
Users can quickly navigate to:
- About
- Services
- Prices
- Book Now

### ℹ️ About Section
Provides information about the business and its commitment to quality laundry services.

### 🖼️ Service Illustration
Displays an image representing laundry services.

### 🧺 Services Offered
- Washing and Drying
- Ironing and Folding
- Dry Cleaning
- Stain Removal
- Express Services
- Special Garment Care

### 💰 Price List
| Service | Price (₹) |
|----------|-----------|
| Washing and Drying | 100 |
| Ironing and Folding | 50 |
| Stain Removal | 50 |

### 📋 Booking Form
Users can enter:
- Name
- Email Address
- Phone Number

### 📞 Contact Information
Business contact email is provided in the footer.

## ▶️ How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/laundry-wallah.git
```

2. Open the project folder.

3. Double-click `index.html`

OR

Open it using a browser:

```bash
start index.html
```

4. The website will open in your default web browser.
